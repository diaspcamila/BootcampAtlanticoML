###### 1. Explique, com suas palavras, o que é machine learning?
- Machine Learning, ou Aprendizado de Máquina, é uma área da computação que envolve o uso de técnicas de treinamento em inteligências artificiais, afim de delas aprenderem algo, como detectar faces, objetos, padrões, entre outros, autonomamente com base em dados e estatísticas.

###### 2. Explique o conceito de conjunto de treinamento, conjunto de validação e conjunto de teste em machine learning.
- Conjunto de treinamento: É o primeiro conjunto que o modelo vai receber, ele serve para ensinar uma base do que ele deve procurar e estudar.
- Conjunto de validação: É usado também durante o treinamento e serve para testes de hiperparâmetros e tipos de modelos diferentes.
- Conjunto de teste: É o conjunto de dados final que o modelo irá receber para entender que ele realmente funciona do jeito correto com a melhor acurácia possível e usá-lo.
###### 3. Explique como você lidaria com dados ausentes em um conjunto de dados de treinamento.
- Caso houver dados ausente no conjunto de dados, procuraria igualar a quantidade entre as classes de alguma forma, ou procurar algum outro conjunto.
###### 4. O que é uma matriz de confusão e como ela é usada para avaliar o desempenho de um modelo preditivo?
- A matriz de confusão representa quais imagens o seu modelo relacionou corretamente ou não a uma classe específica, ou seja se ele tiver acertado várias imagens de pessoas e seus respectivos nomes (classe), a matriz terá uma diagonal principal com valores acrescentados, pois cada linha é uma instância da classe enquanto cada coluna é a previsão da classe.
